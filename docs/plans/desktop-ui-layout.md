# MealMuse PC 端 Desktop 布局设计稿

> 版本：v1.0 | 创建日期：2026-06-07 | 对应：ui-design.md 的 PC 端扩展

---

## 1. 设计原则

### 1.1 响应式策略：Mobile-First + Desktop 增强

- **Mobile (<768px)**：单列，底部导航，信息逐层展开（已有设计）
- **Tablet (768-1024px)**：双列，左侧导航收起为图标，主内容区扩大
- **Desktop (>1024px)**：三栏或双栏，充分利用横向空间，信息平铺展示

### 1.2 PC 端特有的交互模式

| 移动端习惯 | PC 端适配 |
|-----------|----------|
| 底部 Tab 导航 | 左侧固定侧边栏导航 |
| 底部输入框 | 页面内嵌表单或右侧浮窗 |
| 下拉刷新 | 页面级刷新按钮或自动刷新 |
| 全屏弹窗 | 右侧抽屉（Drawer）或模态框 |
| 长按操作 | 右键菜单或悬浮操作按钮 |
| 单列垂直滚动 | 多列并行，关键信息一屏可见 |

### 1.3 信息密度规范

| 元素 | Mobile | Desktop |
|------|--------|---------|
| 卡片宽度 | 100% - 32px padding | 最大 480px，可网格排列 |
| 列表项高度 | 64-72px | 56-64px（鼠标可操作更小区域） |
| 图表尺寸 | 全宽 300px 高 | 可横向并排，400px 高 |
| 边距 | 16px | 24-32px |
| 栅格系统 | 4px 基栅 | 8px 基栅 + 12 列布局 |

## 2. 全局导航结构（Desktop）

### 2.1 顶部栏（Top Bar）

```
┌──────────────────────────────────────────────────────────────────────────────┐
│  🌿 MealMuse                    🔍 全局搜索...           🔔  👤 小美  ▼   │
└──────────────────────────────────────────────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| Logo | 左侧固定，点击返回首页 |
| 全局搜索 | `Cmd/Ctrl + K` 唤起，可搜索食物、食谱、历史记录 |
| 通知中心 | 饮食提醒、家庭动态、AI 建议推送 |
| 用户头像 | 下拉菜单：个人中心 / 设置 / 深色模式 / 退出登录 |

### 2.2 左侧边栏（Side Navigation）

```
┌────────┐
│  🌿    │  <- Logo（折叠时仅显示图标）
├────────┤
│ 🏠 首页 │  <- 当前页高亮（Primary 色左侧 3px 竖线）
│ 📝 记录 │
│ 🍽️ 计划 │
│ 💬 对话 │  <- 未读消息时显示红点数字
│ 📊 报告 │
├────────┤
│ 👤 家庭 │  <- 家庭共享入口（有家庭时显示）
│ 🍎 水果 │  <- 水果专项推荐
├────────┤
│ ⚙️ 设置 │
└────────┘
```

**交互说明：**
- 默认宽度 200px，可折叠为 64px 仅图标模式
- 折叠后鼠标悬浮展开 Tooltip
- 当前页面高亮：左侧 3px Primary 色竖线 + 背景色 `#F0FDF4`
- 支持 `Cmd/Ctrl + 1~6` 快捷键切换页面

## 4. 组件级 Desktop 适配

### 4.1 数据表格（PC 端特有）

家庭共享页、食物库管理等场景需要使用表格：

```
┌──────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
│ 成员      │ 目标      │ 今日热量  │ 蛋白质   │ 碳水     │ 脂肪     │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│ 👤 小美   │ 减脂      │ 860/1580 │ 45/80g   │ 120/200g │ 25/55g   │
│ 👤 阿杰   │ 增肌      │ 2100/2500│ 120/150g │ 280/350g │ 65/80g   │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘
```

- 表头固定，支持横向滚动
- 行悬浮显示操作按钮（编辑/删除）
- 支持列排序

### 4.2 拖拽调整餐食（PC 端增强）

在 Desktop 的餐食计划页：
- 支持拖拽食物项在三餐之间移动
- 拖拽时显示营养变化预览
- 右键食物项显示"替换/删除/收藏"

### 4.3 快捷键体系

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + K` | 全局搜索 |
| `Cmd/Ctrl + 1` | 跳转首页 |
| `Cmd/Ctrl + 2` | 跳转记录 |
| `Cmd/Ctrl + 3` | 跳转计划 |
| `Cmd/Ctrl + 4` | 跳转对话 |
| `Cmd/Ctrl + 5` | 跳转报告 |
| `Cmd/Ctrl + Shift + R` | 快速记录（唤起输入浮窗）|
| `Esc` | 关闭弹窗/抽屉 |
