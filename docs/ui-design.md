### 1.2 字体规范

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 28px | Bold (700) | 页面标题 |
| H2 | 22px | Semibold (600) | 区块标题 |
| H3 | 18px | Semibold (600) | 卡片标题 |
| Body | 16px | Regular (400) | 正文 |
| Caption | 14px | Regular (400) | 辅助文字 |
| Small | 12px | Regular (400) | 标签、时间 |

### 1.3 间距系统

基于 4px 网格：4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48 / 64

### 1.4 圆角

| 元素 | 圆角 |
|------|------|
| 按钮 | 8px |
| 卡片 | 12px |
| 输入框 | 8px |
| 头像 | 50% |
| 底部弹窗 | 16px (顶部) |

## 4. 交互说明

### 4.1 动效规范

| 交互 | 动效 | 时长 |
|------|------|------|
| 页面切换 | 淡入淡出 | 200ms |
| 卡片出现 | 从下方滑入 | 300ms |
| 按钮点击 | 缩放 0.95 | 100ms |
| 加载状态 | 骨架屏 + 脉冲动画 | - |
| AI 打字 | 逐字显示 | 30ms/字 |
| 营养进度条 | 从左到右填充 | 500ms |

### 4.2 状态设计

| 场景 | 状态 | 表现 |
|------|------|------|
| 数据加载中 | Loading | 骨架屏 |
| 数据为空 | Empty | 插图 + 引导文案 |
| 网络错误 | Error | 重试按钮 |
| 操作成功 | Success | Toast 提示 |
| AI 回复中 | Streaming | 打字机效果 |

### 4.3 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| Mobile | < 768px | 单列，底部导航 |
| Tablet | 768-1024px | 双列，侧边导航 |
| Desktop | > 1024px | 三列，侧边导航 |
