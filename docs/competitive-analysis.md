## 3. 关键洞察

### 3.1 市场空白 — 没人做好的事

| 空白 | 说明 | MealMuse 机会 |
|------|------|--------------|
| **生理期 × 饮食规划** | 美柚有生理期但不做餐食规划；薄荷有餐食但不管生理期 | 结合两者，周期感知的三餐推荐 |
| **夫妻/家庭共享** | 所有饮食 App 都是单用户模式 | 家庭账户，一人做饭全家受益 |
| **水果专项推荐** | 水果只是食物库里的一行，没人专门做推荐 | 按季节/体质/目标推荐水果 |
| **AI 深度对话** | 现有 App 的 AI 只是关键词匹配 | 基于真实饮食数据的深度对话 |

### 3.2 可借鉴的最佳实践

| 来源 | 功能 | 借鉴方式 |
|------|------|----------|
| **薄荷健康** | 本地化食物库（数十万条） | AI 解析 + 预置食物库双引擎 |
| **美柚** | 经期四阶段饮食建议 | 内置经期阶段模型，自动调整餐食推荐 |
| **美柚** | 16 维度个性化 | 建立用户画像多维度标签体系 |
| **Keep** | 热量摄入 vs 消耗 | 后续接入运动数据，展示热量缺口 |
| **PlateJoy** | 家庭偏好过滤 | 家庭成员各自偏好，生成全家餐方案 |
| **PlateJoy** | 购物清单 | 餐食计划自动生成购物清单 |
| **MyFitnessPal** | 条码扫描 | 后续支持扫描包装食品 |

## 4. MealMuse 差异化定位

### 核心定位

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   MealMuse = AI 饮食助手 × 女性健康管理 × 家庭共享          │
│                                                             │
│   别人做的：记录热量 ← 我们做的是：AI 帮你想吃什么           │
│   别人做的：单人使用 ← 我们做的是：夫妻/家庭一起用           │
│   别人做的：通用推荐 ← 我们做的是：生理期/备孕/体质感知      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 一句话差异化

> **MealMuse 是唯一一款能感知女性生理周期、支持夫妻/家庭共享、并提供 AI 深度对话的智能饮食助手。**
