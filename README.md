# 🍽️ MealMuse - AI 智能饮食管理助手

> **每一餐，都是对自己的善待**

MealMuse 是一款 AI 驱动的智能饮食管理应用，帮助你轻松记录饮食、获取个性化营养建议、追踪健康数据，让每一餐都吃得更科学、更健康。

---

## ✨ 核心功能

### 📝 智能饮食记录
- **自然语言输入**：用日常语言描述你吃了什么，AI 自动解析食物和营养成分
- **拍照识别**（开发中）：拍一张照片，AI 识别食物并记录
- **每日三餐追踪**：自动计算早/中/晚餐的热量和营养素摄入
- **营养评分**：直观展示每日营养摄入是否达标

### 🤖 AI 营养师对话
- **专业问答**：随时向 AI 营养师提问饮食健康问题
- **个性化建议**：根据你的身体数据和饮食习惯给出定制建议
- **节气养生**：结合时令节气推荐应季食材和食疗方案
- **快捷问题**：预设常见问题，一键获取答案

### 📊 健康报告分析
- **每日报告**：今日热量、营养素摄入详情
- **每周报告**：一周饮食趋势分析和改进建议
- **营养雷达图**：六大营养素（蛋白质、碳水、脂肪、维生素、矿物质、膳食纤维）可视化展示
- **历史数据**：随时查看过往饮食记录和健康趋势

### 🍳 个性化餐食计划
- **AI 生成三餐**：根据你的健康目标自动生成每日餐食方案
- **一键替换**：不喜欢某道菜？点击"换一个"立即生成新方案
- **目标导向**：支持减脂、增肌、备孕、养生等多种健康目标
- **偏好适配**：考虑你的口味偏好、过敏原、饮食禁忌

### 🩸 经期感知饮食（女性专属）
- **经期记录**：轻松记录生理周期
- **四阶段追踪**：自动识别月经期、卵泡期、排卵期、黄体期
- **阶段饮食建议**：不同阶段推荐不同的营养补充方案
- **水果推荐**：经期适宜和应避免的水果清单

### 👨‍👩‍👧‍👦 家庭共享
- **创建家庭**：邀请家人加入，共享饮食管理
- **成员管理**：查看家人的饮食记录和健康数据
- **共享计划**：为全家人制定统一的餐食计划
- **邀请码机制**：通过邀请码快速添加家庭成员

### 💪 身体数据管理
- **体重记录**：记录体重变化，生成趋势图表
- **体脂监测**：追踪体脂率变化
- **睡眠记录**：记录每日睡眠时长和质量
- **运动数据**：记录步数、运动时长等数据
- **血压血糖**：支持血压、血糖等健康指标记录

### 👤 个人中心
- **健康档案**：设置身高、体重、年龄等基础信息
- **健康目标**：选择你的健康管理目标
- **饮食偏好**：设置口味偏好、过敏原、不喜欢的食物
- **账号管理**：修改个人信息、退出登录

## 🚀 快速开始

### 环境要求

**前端**
- Node.js 18+
- npm / yarn / pnpm

**后端**
- Python 3.11+
- PostgreSQL 16+
- Redis 7+

### 安装步骤

#### 1. 克隆项目

```bash
git clone <repository-url>
cd meal-muse
```

#### 2. 启动后端

```bash
cd meal-muse-api

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，配置数据库和 Redis 连接

# 启动服务
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

#### 3. 启动前端

```bash
cd meal-muse-web

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

#### 4. 访问应用

- 前端：http://localhost:3000
- 后端 API 文档：http://localhost:8000/docs
- 开发环境验证码：`888888`

## 🔧 技术架构

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   Frontend      │     │   Backend       │     │   Database      │
│   Next.js 16    │────▶│   FastAPI       │────▶│   PostgreSQL    │
│   React 19      │     │   SQLAlchemy    │     │   Redis         │
│   Tailwind CSS  │     │   通义千问 AI   │     │                 │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

**前端技术栈**
- Next.js 16 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4
- Zustand 5 (状态管理)
- Recharts (图表)
- Lucide React (图标)

**后端技术栈**
- Python FastAPI
- SQLAlchemy 2.0 (异步 ORM)
- PostgreSQL 16
- Redis (缓存/验证码)
- JWT 认证
- 通义千问 qwen-plus (AI 模型)

## 📖 API 接口

### 认证模块
- `POST /api/v1/auth/send-code` - 发送验证码
- `POST /api/v1/auth/register` - 用户注册
- `POST /api/v1/auth/login` - 用户登录

### 饮食记录
- `POST /api/v1/diet/records` - 创建饮食记录
- `GET /api/v1/diet/records` - 获取饮食记录列表
- `GET /api/v1/diet/today` - 获取今日记录
- `DELETE /api/v1/diet/records/{id}` - 删除记录

### 餐食计划
- `POST /api/v1/meals/generate` - AI 生成餐食计划
- `GET /api/v1/meals/plan` - 获取当前计划
- `POST /api/v1/meals/{id}/replace` - 替换菜品

### AI 对话
- `POST /api/v1/chat/send` - 发送消息给 AI

### 健康报告
- `GET /api/v1/reports/daily` - 获取每日报告
- `GET /api/v1/reports/weekly` - 获取每周报告
- `GET /api/v1/reports/nutrition-radar` - 获取营养雷达图

### 经期追踪
- `POST /api/v1/menstrual/records` - 记录经期
- `GET /api/v1/menstrual/current` - 获取当前阶段
- `GET /api/v1/menstrual/records` - 获取经期历史

### 家庭共享
- `POST /api/v1/family/create` - 创建家庭
- `POST /api/v1/family/join` - 加入家庭
- `GET /api/v1/family/my` - 获取我的家庭
- `DELETE /api/v1/family/members/{id}` - 移除成员

### 身体数据
- `POST /api/v1/body/metrics` - 记录身体数据
- `GET /api/v1/body/metrics` - 获取历史数据
- `GET /api/v1/body/metrics/latest` - 获取最新数据

## 💡 使用技巧

1. **首次使用**：先在「个人中心」完善身高、体重等基础信息，AI 建议会更精准
2. **记录饮食**：用自然语言描述即可，如"早上吃了一个包子和一杯豆浆"
3. **获取计划**：点击「计划」页面的"生成今日三餐"，AI 会根据你的目标定制餐单
4. **查看报告**：定期查看「报告」页面，了解营养摄入是否均衡
5. **经期管理**：女性用户可在「经期」页面记录周期，获取阶段专属饮食建议
